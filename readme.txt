
# Bulk posting in Facebook groups throw Python selenium.

As facebook API didn't acess to publish the post throw API. This python script automatically post video,image or text as per your requirement.



## Then install the all the requirements by runing the following commands

```bash
  python -m pip install requirements.txt
```

`user=`

`video_or_image_absolute_path=`

## Execute main.py file
Finally, run the following command

